# Firmixについて

## 概要

Firmixは、マイコンのファームウェアを投稿して公開できるプラットフォームです。

ファームウェアのコンフィグレーションの機構を提供しており、ファイルのダウンロード時に事前に定義されたパタメータを注入することができます。

自作キーボードシステム[Kermite](https://kermite.org)のカスタムファームウェアを作って配布するためのサービス、という位置づけをメインに開発を行っています。

ファームウェアの動的な構成の仕組みには汎用性を持たせてあり、キーボード以外の電子工作のファームウェアにも応用できるようにしました。



## 主な機能

FirmixのWebサービスには以下の機能があります。

- Githubアカウントでのログイン
- プロジェクトの一覧と詳細の表示
- ファームウェアのコンフィグレーション
- プロジェクトの管理
- ローカルプロジェクトの表示確認とコンフィグレーション
- ローカルプロジェクトの投稿
- Github Actionsなどから利用できるAPIによる投稿

## 開発環境

### プロジェクト

Firmixでは、プロジェクトという単位で投稿内容を扱っています。  
ビルドツールや依存パッケージの管理がしやすい、PlatformIOを利用した構成を採用しています。

### MCU

- RP2040
- ATSAMD21 (対応予定)
- nRF52840 (対応予定)

RP2040に対応しています。  
今後、UF2ファイルでファームウェアを書き込める他のMCU(ATSAMD21, nRF52840)にも対応する予定です。
