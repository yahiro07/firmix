# ファームウェアの書き込み

Firmixでは、公開されているファームウェアをダウンロードして、ボードに書き込む事ができます。
ここではファームウェアの書き込みについてチュートリアル形式で説明します。

## サンプルプロジェクト

ここでは、以下のプロジェクトをターゲットにします。  
RP2040のボード上のLEDが光る、動作確認用のプログラムです。

## ボードの準備

以下のボードのどれかを用意してください

- Raspberry Pi Pico
- KB2040
- Xiao RP2040
- Tiny2040
- RP2040-Zero
- ProMicro RP2040

## フォームの入力

このプロジェクトでは、ボードの種類を数値で指定するように構成されています。
お手元のボードに合わせて、値を入力してください。

|ボード|値|
|:--|:--:|
|Raspberry Pi Pico|1|
|KB2040|2|
|Xiao RP2040|3|
|Tiny2040|4|
|RP2040-Zero|5|
|ProMicro RP2040|6|


## 書き込み

### ファームウェアの生成
「UF2ダウンロード」のボタンを押すと、フォームの入力内容を反映したファームウェアファイルが生成されます。
ファイルの保存先を選ぶダイアログが表示されます。

### ボードのリセット
ボードをリセットしてブートローダモードにします。Raspberry Pi Picoの場合はBOOTボタンを押しながらUSBポートに接続してください。
それ以外のボードでは、ボードをUSBポートに繋いだ状態でBOOTボタンを押しながら、リセットボタンを押して離します。

### 書き込み
ボードに対応する外部ドライブが現れます。ダイアログ上でドライブを選択してそこにUF2ファイルを保存します。

## 動作の確認

ボード上のLEDが点滅すればOKです。フルカラーLEDを搭載しているボードでは、色が変わりながらLEDが光ります。
