# 一体型キーボードサンプル

一体型のキーボードファームウェアのサンプルです。

Kermiteに対応しています。
Kermite対応のキーボードとして動作する、最低限のものです。
ボードLEDとキーマトリクスの機能があります。

## ボードLED

ボードと設定値の対応は以下のとおりです

|値|ボード|
|--|--|
|0|なし|
|1|RPiPico|
|2|KB2040|
|3|Xiao RP2040|
|4|RP2040-Zero|
|5|Tiny2040|
|6|ProMicro RP2040|


## キーマトリクス

行/列それぞれ最大16個までのピンを指定できます。
